Archivo BAT: Chiste

Este archivo BAT presenta una serie de mensajes para contar un chiste en actos usando comandos de la línea de comandos.

@ECHO OFF

El comando @ECHO OFF oculta los comandos mientras se ejecutan en la consola, mostrando solo el texto deseado. El símbolo @ al inicio hace que este comando también se oculte.

ECHO

El comando ECHO muestra mensajes en la consola. Aquí, se usa para mostrar cada acto del chiste:

ECHO PRIMER ACTO: SALE UN "PAVO" CON UN CUCHILLO.
ECHO SEGUNDO ACTO: SALE OTRO "PAVO" CON UNA PISTOLA DE RAYOS LASER.
ECHO TERCER ACTO: SALE OTRO "PAVO" CON UNA ESPADA.
ECHO COMO SE LLAMA LA OBRA?
ECHO LOS PAVOS RANGERS.

PAUSE

El comando PAUSE detiene la ejecución del archivo hasta que el usuario presiona una tecla. En este archivo, cada acto se pausa para que el usuario pueda leer el mensaje antes de pasar al siguiente.

CLS

El comando CLS limpia la pantalla de la consola, eliminando el texto anterior. Después de cada pausa, se limpia la pantalla para mostrar un "acto" en una pantalla limpia.
