Comandos FOR e IF en Archivos BAT

Los comandos FOR e IF son herramientas esenciales en los archivos BAT que permiten controlar la ejecución de scripts mediante bucles y condiciones, respectivamente. Estos comandos se utilizan para realizar tareas repetitivas y condicionales de forma eficiente.

Comando FOR

El comando FOR permite repetir un conjunto de instrucciones para cada elemento en una lista o rango especificado. Se utiliza para iterar sobre archivos, directorios o listas de valores.

Ejemplo básico del comando FOR, que lista archivos con extensión `.txt` en el directorio actual:

FOR %%F IN (*.txt) DO ECHO Archivo encontrado: %%F

Este ejemplo recorrerá todos los archivos `.txt` en el directorio y mostrará cada archivo con el mensaje Archivo encontrado:.

Ejemplo de uso de FOR con un rango de números:

FOR /L %%N IN (1, 1, 5) DO ECHO Número actual: %%N

El parámetro /L permite especificar un rango (en este caso, de 1 a 5). Cada número del rango se mostrará con el mensaje Número actual:.

Comando IF

El comando IF permite ejecutar comandos de manera condicional en función de valores específicos. Se utiliza para comparar valores y tomar decisiones en el flujo del script.

Ejemplo básico de uso del comando IF para comparar dos valores:

SET /A VALOR=5
IF %VALOR% GEQ 5 ECHO El valor es mayor o igual a 5

En este caso, el comando ECHO se ejecutará solo si VALOR es mayor o igual a 5. GEQ significa "greater or equal" (mayor o igual).

Ejemplo de uso de IF con la existencia de archivos:

IF EXIST "archivo.txt" (
    ECHO El archivo existe.
) ELSE (
    ECHO El archivo no existe.
)

Este ejemplo verifica si el archivo archivo.txt existe en el directorio actual. Si existe, muestra el mensaje El archivo existe.; si no, muestra El archivo no existe.

Combinando FOR e IF

También es posible combinar los comandos FOR e IF para realizar tareas más avanzadas. Por ejemplo, podemos iterar sobre una lista de archivos y verificar si cumplen con cierta condición.

Ejemplo que muestra solo los archivos con tamaño mayor a 0 bytes en el directorio actual:

FOR %%F IN (*.*) DO (
    IF %%~zF GTR 0 (
        ECHO Archivo con contenido: %%F
    )
)

El modificador %%~zF obtiene el tamaño del archivo %%F. Si el tamaño es mayor a 0 (GTR significa "greater than"), muestra el nombre del archivo.
